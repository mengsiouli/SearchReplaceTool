# SearchReplaceTool
**SearchReplaceTool** 是一個簡單的文字搜尋與取代工具，使用 C# WinForms 開發。  

 ## 功能
- 在文字方塊中搜尋指定文字
- 對搜尋到的文字進行取代
- 支援逐一確認是否替換
- 自動跳到下一個搜尋結果
- 提供錯誤提示，若找不到匹配文字會提醒使用者

## 使用方式
1. 在「文件內容」輸入要搜尋的文字內容
2. 在「搜尋文字」輸入想查找的文字
3. 點擊 **搜尋** 按鈕尋找下一個匹配文字
4. 點擊 **取代** 按鈕，系統會詢問是否替換目前匹配文字
5. 替換後自動跳到下一個匹配文字，直到搜尋結束

## 單元測試
本專案提供，使用 **MSTest** 框架進行單元測試，主要驗證：
- **搜尋文字 (`SearchText`)**  
  測試不同起始索引與搜尋結果，包含找得到與找不到的情況。
- **取代文字 (`ReplaceText`)**  
  測試不同起始索引、替換文字與空字串的情況。
- **輸入驗證 (`CheckDocInputAndSearchText`)**  
  驗證文件內容與搜尋文字是否為空或有效。

測試可確保文字搜尋與取代邏輯正確運作。
